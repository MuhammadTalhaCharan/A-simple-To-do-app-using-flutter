# Week 2 Flutter Assignment

## Overview

This application demonstrates:

* State Management using setState()
* Persistent Storage using SharedPreferences
* To-Do List Management
* Local Data Saving and Retrieval

## Features

### To-Do List

* Add new tasks
* Display tasks in ListView
* Delete tasks
* Automatically save tasks

### Persistent Storage

* Tasks remain available after app restart
* Data stored locally using SharedPreferences

### State Management

* Dynamic UI updates using setState()

## Technologies Used

* Flutter
* Dart
* SharedPreferences

## How to Run

1. Clone repository
2. Run flutter pub get
3. Run flutter run
